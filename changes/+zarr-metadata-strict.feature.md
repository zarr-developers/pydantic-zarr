Added strict Zarr v3 specs (importable from `pydantic_zarr.v3`): `StrictArraySpec` (a directly constructible class with runtime fill-value validation), public per-dtype classes (`BoolArraySpec`, `Int8ArraySpec`, `Float64ArraySpec`, etc.) with precise static `fill_value` types, `AnyStrictArraySpec` (the discriminated union for validating `zarr.json` dicts via `TypeAdapter`), and `StrictGroupSpec` (group counterpart requiring all members to be recursively strict). `fill_value` is coupled to `data_type` (e.g. `float64` accepts `"NaN"` but `int64` does not).
